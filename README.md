# Heart Disease Prediction (Machine Learning)

This project builds a machine learning model to predict the likelihood of heart disease using patient health data from the Cleveland dataset.

## Key Features
- Logistic Regression model using scikit-learn  
- Data preprocessing (missing values, encoding, scaling)  
- Real-time user input for predictions  
- Model accuracy of ~85%  

## Tools
Python, Pandas, NumPy, Scikit-learn

## Results
The model effectively classifies patients into high-risk and low-risk categories, demonstrating the potential of machine learning in healthcare prediction.

## Files
 `model_training.py` – model development and training  
- `data_preprocessing.py` – data cleaning and transformation  
- `heart_disease_prediction.pdf` – full project report
